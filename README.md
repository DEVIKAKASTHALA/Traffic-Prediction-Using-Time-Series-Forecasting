# Traffic-Prediction-Using-Time-Series-Forecasting
This project predicts future traffic volume using time series forecasting techniques. The ARIMA model is used to analyze historical traffic data and generate predictions.
## Technologies Used
- Python
- Pandas
- NumPy
- Statsmodels (ARIMA)

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the project:
   python main.py

## Output
The system predicts future traffic volume based on historical data.
